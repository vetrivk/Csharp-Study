C# Document & Interview Questions

1.What is a C#?

C# is a Programming Language developed by Microsoft within its .NET framework. C# is an object-oriented-programming language used to building a variety of applications include desktop, Web applications, Games, Mobile apps and more.

Below is a simple C# code example of printing a message 
       
        Using System;
        namespace your_project_name // Organizes the code.
        {
            class Program //Contains the program's main logic
            {
                static void Main(string[] args) //Entry point of the application
                {
                  Console.WriteLine("Hello World"); //The cursor moves to the next line

                  Console.Write("I ");
                  Console.Write("am ");
                  Console.Write("Student"); //The cursor does not move to the next line

                  Console.ReadKey(); //Waits for a key press to keep the console window open
                }
            }
        }
OutPut:

Hello World

I am Student
